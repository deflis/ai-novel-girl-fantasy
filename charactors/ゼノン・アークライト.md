
# 1. 基本情報

- 名前: ゼノン・アークライト
- 年齢: 20歳
- 立場: 宮廷魔導師団・第一席（攻撃魔術専門）。アリアの元・兄弟子（あるいは同期）。

## 外見

- 夜空のような黒髪と、冷ややかなアメジストの瞳。
- 仕立ての良い黒基調の宮廷魔導師服を隙なく着こなす。
- 一見完璧なエリートだが、どこか影があり疲れた表情を見せることがある。

## 性格

- 表: クールで合理的。皮肉屋。宮廷の規律と成果を重視する現実主義者。
- 裏: 一途で執着心が強い。一度惹かれたものは手放さない、不器用な情深さを持つ。

# 2. アリアとの関係性 — 『歪んだ執着』と『すれ違い』

ゼノンは、アリアが宮廷を辞めた「本当の理由」を知らないか、あるいは違う解釈で受け取っています。

- 唯一のライバル兼理解者（過去）:
	- 二人は同じ師匠の下で学んだ兄弟子（ゼノン）と妹弟子（アリア）。
	- ゼノンは攻撃魔術、アリアは魔法薬学を専門とし、互いの才能を認め合っていた。
	- 価値観の違い（力を重んじるゼノンと癒しを重んじるアリア）は、しばしば議論を生んだ。
	- アリア側の認識: 「口うるさいが尊敬できる兄弟子」。
	- ゼノンの本心: アリアの才能に焦燥と嫉妬を覚える一方で、「隣に立つのは彼女だけだ」という独占的な想いを抱く。

- 決定的なすれ違い:
	- アリアが『自白剤』の開発を強要され苦悩していた時、ゼノンは長期討伐任務で王都を離れていた。
	- アリアは彼に相談できず、辞表を出して出奔する。
	- 任務から戻ったゼノンは、アリアが『国家機密を持ち出した逃亡者』として指名手配されていることを知る。

# 3. 刺客としての動機

- 表向きの任務: 国家重要機密保持者（アリア）の確保、および反逆の証拠（聖獣との接触）の調査。

- ゼノンの本心:
	- 裏切りへの怒り: 「なぜ俺に一言も相談しなかったのか」「なぜ俺たちを捨てたのか」。
	- 才能への執着: 「その才能を辺境で腐らせるわけにはいかない。お前が作る薬は宮廷――俺たちに必要だ」。
	- 歪んだ好意: 「お前を連れ戻せるのは俺だけだ。お前の隣は俺の場所だ」という独占欲。

# 4. 再会と今後の展開

## 再会（例）

- アリアのカフェが宮廷に『奇跡の店』として報告され、ゼノンが調査の名目で辺境へ派遣される。
- お忍びでカフェを訪れ、客としてアリアの『目覚めのハーブティー』を飲む。
- 一口で、それが宮廷時代の研究（アリアが蔑ろにされた研究）の応用だと気づく。
- ゼノン: 「……こんなものを作るために、お前は宮廷を捨てたのか、アリア」

## 聖獣との対立

- ゼノンは強引にアリアを連れ戻そうと魔術を行使するが、用心棒のグリフォン（通称: グリさん）が立ちはだかる。
- 宮廷最強クラスのゼノンと伝説の聖獣が一触即発の緊張状態に。
- アリアは「私はもうあなたの知っているアリアじゃない。私のスローライフを邪魔しないで！」と叫び、聖獣たちと共に彼を撃退する。

## 恋愛要素の進展

- ゼノンはアリアが聖獣を手懐けていることに衝撃を受け、任務の難易度と執着が高まる。
- 王都へ戻らず辺境に滞在し、アリアのカフェを監視（ストーキング）するようになる。
- 監視中、アリアの穏やかな笑顔や、聖獣たちに懐く姿を見て、ゼノンは彼女の知らなかった一面に動揺する。
- アリアもまた、ゼノンが宮廷で苦労している様子や、追手でありながら非情になりきれない不器用な優しさに気づき、関係が複雑化していく。
