# 1. 基本情報

- 名前: アリア・レンフィールド
- 年齢: 17歳
- 外見: 落ち着いたアッシュブロンドの髪。宮廷時代はきっちり編み込んでいたが、今は緩くまとめている。瞳は知的なサファイアブルー。華奢だが、長年の研究で鍛えられた集中力と体力を持つ。
- 性格: 基本は温厚でマイペース。探究心が強く、一度没頭すると周りが見えなくなる（特に魔法薬と食材のこと）。宮廷では「沈黙の調合師」と呼ばれていたが、無口なのではなく「中身のない政治的な会話」が苦手なだけ。

# 2. 宮廷魔導師としての才能と実績

- 専門分野: 「魔法薬学（ポーション）」および「付与魔術」。
- 才能の源泉: 複数の魔術効果を一つの液体（薬）や物体（菓子）の中で、反発させずに安定させる「調合」と「定着」のセンスがずば抜けている。

実績（なぜエリートか）:

15歳で宮廷魔導師見習いになる。

16歳の時、従来の治癒薬とマナ回復薬を完全融合させた「戦場用ハイブリッド・ポーション」を開発。これにより、負傷兵の生存率と魔導師部隊の継戦能力が飛躍的に向上し、最年少で正式な宮廷魔導師（第三席）に抜擢される。

# 3. 宮廷で疲弊した理由（具体例）

彼女が疲弊したのは、単なる激務ではなく、彼女の信念と宮廷の体質が根本的にミスマッチだったためです。

① 成果主義と「効率」の強要:

宮廷が彼女に求めたのは「より安く、より早く、より強力な」軍事用・政治用の魔法薬でした。
彼女が研究していた「飲むと心が安らぐハーブティー（副次効果：軽度の精神安定）」などは、「即効性がない」「金にならない」と切り捨てられ、研究予算も回されませんでした。

② 派閥争いと嫉妬:

彼女は最年少のエリートだったため、古参の貴族派閥（攻撃魔術至上主義）から「薬屋風情が」と見下され、同時にその才能を恐れられていました。
彼女の「ハイブリッド・ポーション」の功績も、直属の上司である貴族に横取りされそうになるなど、日常的に妨害を受けていました。

③ 辞表を叩きつけた「決定的な事件」:

対立国との緊張が高まる中、上層部から**「自白効果のある（＝精神に作用する）無味無臭の魔法薬」**の開発を暗に命じられます。
それは「人を幸せにする」という彼女の信念（後述）から最も遠い、「人の尊厳を奪う」ための薬でした。
「私が作りたいのは、こんなものじゃない！」と、研究資料を（上司の顔に）叩きつけ、そのまま王城を飛び出しました。

# 4. 趣味（お菓子作り）との関連性

- 原体験: 幼い頃、魔導師だった祖母が作ってくれた「飲むとぐっすり眠れるホットミルク（安眠の魔法薬入り）」や「食べると元気が出るハチミツクッキー（疲労回復の付与魔術）」が、彼女の原点です。
- 信念: 「魔法は、日常を少しだけ幸せにするためにある」
- 宮廷でのストレス発散: 宮廷では、厳格な管理下で「薬」しか作れませんでした。その反動で、宿舎の自室では夜な夜な趣味のお菓子作りに没頭。その際、「苦い精神安定薬」をラベンダー風味のクッキーに練り込んだり、「マナ回復ポーション」を飲みやすいフルーツコンポートにしたりと、趣味と実益を兼ねた研究を秘密裏に行っていました。
- カフェへの着想: 宮廷を辞めると決めた時、彼女に残ったのは「公式な魔法薬学の知識」と「異端とされたお菓子作りの技術」でした。
「宮廷ではだめだったけど、私が本当に作りたかった『人を笑顔にする魔法薬（お菓子）』なら、どこかで誰かの役に立つかもしれない」
これが「魔法薬カフェ」の原点となります。

# 5. 辺境を選んだ理由

- ① 祖母の故郷: 彼女の原点である祖母が住んでいた、自然豊かな土地（の近く）。
- ② 素材の宝庫: 宮廷では手に入りにくかった、魔力を豊富に含んだ良質なハーブや清らかな水が手に入る場所。
- ③ 匿名性: 首都から遠く、宮廷の権力争いと無縁な場所。「宮廷魔導師第三席アリア」ではなく、ただの「カフェ店主アリア」として再スタートできる場所を求めました。
