# 1. 基本情報

- 名前: アリア・レンフィールド
- 年齢: 17歳（15歳で見習い魔導師、2年間勤務後に退職、3ヶ月前に辺境へ、現在カフェ開店1ヶ月）
- 出身: 村の小さな薬師の家
- 外見: 落ち着いたアッシュブロンドの髪。宮廷時代はきっちり編み込んでいたが、今は緩くまとめている。瞳は知的なサファイアブルー。華奢だが、長年の研究で鍛えられた集中力と体力を持つ。
- 魔力属性: 水と風の混合（珍しいタイプ）
- 性格: 基本は温厚でマイペース。探究心が強く、一度没頭すると周りが見えなくなる（特に魔法薬と食材のこと）。宮廷では「沈黙の調合師」と呼ばれていたが、無口なのではなく「中身のない政治的な会話」が苦手なだけ。心を込めて丁寧に物事を行う性格で、それが宮廷では「効率が悪い」と叱責される原因となった。

# 2. 宮廷魔導師としての才能と実績

- 専門分野: 「魔法薬学（ポーション）」および「付与魔術」。
- 才能の源泉: 複数の魔術効果を一つの液体（薬）や物体（菓子）の中で、反発させずに安定させる「調合」と「定着」のセンスがずば抜けている。祖母から受け継いだ「心を込めて丁寧に作る」技術により、マニュアル通りでも基準を上回る品質を実現できる。
- 魔力制御の特徴: ゼノンから「定着の技術が粗い」と指摘され、「螺旋を描くように定着させる」技術を学ぶ。これにより魔力の揮発率が下がり、ポーションの持続時間が伸びるようになった。

実績（なぜエリートか）:

- 15歳で史上最年少の宮廷魔導師見習いに抜擢される。
- 見習い初日から、基準を大幅に上回る品質の治癒薬を量産（ただし「均一性を乱す」として叱責される）
- 卓越した調合技術により、軍事用ポーションの品質向上に貢献
  - ただし、品質が高すぎると「均一性を乱す」として叱責されるという矛盾に苦しむ

# 3. 宮廷で疲弊した理由（時系列）

彼女が疲弊したのは、単なる激務ではなく、彼女の信念と宮廷の体質が根本的にミスマッチだったためです。

## タイムライン（宮廷在籍2年間）

### 第2話：入宮初日（15歳、2年前）
- 「人を幸せにする魔法薬」を作る夢を抱いて宮廷へ
- 配属されたのは軍事用ポーションの量産ライン
- 上司グレン主任から「効率」「量産」「納期厳守」を強要される
- 1日50本のノルマ、達成できないと減給
- 祖母の教え「魔法薬は、心を込めて」を胸に、丁寧に作業を続ける

### 第3話：ゼノンとの出会い（入宮3日後）
- 宮廷魔導師第一席ゼノン・アークライトと出会う
- アリアの作った治癒薬の品質を一目で見抜き、「定着の応用」技術を教える
- 夜間に共同研究を始める（お互いの専門分野を補完し合う関係）
- 初めて宮廷に居場所ができたと感じる

### 第4話：派閥の壁（入宮3週間後）
- 夜間研究が貴族派閥の上級魔導師たちに発覚
- 「見習いが勝手な研究をすることは規則違反」として夜間研究を禁止される
- 品質が高すぎることも「均一性を乱す」として叱責される
- ゼノンから「耐えろ。いつか実力が誰にも否定できないほどになれば状況は変わる」と助言される

### 第5話：蔑ろにされる想い（夜間研究禁止から1週間後）
- 諦めずに、私費で安眠効果のあるポーションを作り始める
- 見習い魔導師セシルや下級兵士たちに配布し、好評を得る
- 上司グレンから「兵士は戦うための道具だ。安眠など不要」と叱責され、配布を禁止される
- ゼノンから「お前は優しすぎる。余計なことをするな」と忠告される

### 第6話：形見の行方（安眠ポーション事件から半年後）
- 研究室が荒らされ、祖母の形見のレシピノートが破り捨てられる
- 貴族派閥による嫌がらせだが、証拠がなく何もできない
- ゼノンが犯人を探すと言うが、事態悪化を恐れて止める
- ゼノンから新しいノートを贈られ、記憶を辿りながら祖母のレシピを書き写し始める

### 第7話：壊れていく心（レシピノート破棄から2週間後、4ヶ月前）
- 激務と睡眠不足で心身ともに限界を迎える（朝6時起床、夜9時帰宅、睡眠4〜5時間）
- 量産作業中にミスが増え、グレンから叱責される日々
- ついに量産作業中に倒れ、医務室送りに
- ゼノンが看病し、ノルマを代わりにこなしてくれる
- ゼノンから「本当に辛くなったら、逃げろ。俺が許す」と告げられる

### 第8話：決断の日（過労で倒れてから1週間後、3ヶ月前）
- 「自白誘発薬」の開発命令が下される（機密指定、期限1ヶ月、無味無臭）
- 拒否すれば降格・宮廷追放
- 3日間悩んだ末、白紙の報告書と辞表を上司グレンに叩きつける
- 「人の尊厳を踏みにじる薬なんて、私には作れません」と宣言
- ゼノンから旅費を受け取り、「お前は、お前の信じる道を行け」と背中を押される
- 宮廷魔導師の制服を置いて、宮廷を去る

# 4. 祖母との思い出とアリアの原点

- **原体験**: 幼い頃、村の薬師だった祖母が作ってくれた温かい魔法薬
  - 「飲むとぐっすり眠れるホットミルク（安眠の魔法薬入り）」
  - 「食べると元気が出るハチミツクッキー（疲労回復の付与魔術）」
  - 「風邪の時のハーブスープ（栄養たっぷり）」
- **祖母の教え**: 「魔法薬は、心を込めて」「魔法は、人を幸せにするためにあるのよ」
- **形見のレシピノート**: 祖母の手書きの文字で、様々な魔法薬とお菓子のレシピが記されていた
  - 宮廷時代、貴族派閥の嫌がらせで破り捨てられる
  - ゼノンから新しいノートを贈られ、記憶を辿りながら書き写し始める
  - 「誰にも奪われない宝物」として、現在も大切に保管している

# 5. 趣味（お菓子作り）との関連性

- **信念**: 「魔法は、日常を少しだけ幸せにするためにある」「誰かの笑顔を作るため」
- **宮廷での密かな研究**:
  - 宿舎の自室に簡易的な調合道具を持ち込み、夜な夜なお菓子作りに没頭
  - 安眠効果のあるラベンダークッキーを焼いたり、疲労回復のハチミツレモネードを作ったり
  - 試作品を見習い魔導師セシルに分け与え、喜ばれる
  - しかし「勝手な研究」として禁止される
- **カフェへの着想**:
  - 宮廷を辞めると決めた時、彼女に残ったのは「公式な魔法薬学の知識」と「祖母から学んだお菓子作りの技術」
  - 「宮廷ではだめだったけど、私が本当に作りたかった『人を笑顔にする魔法薬（お菓子）』なら、どこかで誰かの役に立つかもしれない」
  - これが「魔法薬カフェ」の原点となった

# 6. 辺境を選んだ理由

- ① 祖母の故郷: 彼女の原点である祖母が住んでいた、自然豊かな土地（の近く）。
- ② 素材の宝庫: 宮廷では手に入りにくかった、魔力を豊富に含んだ良質なハーブや清らかな水が手に入る場所。
- ③ 匿名性: 首都から遠く、宮廷の権力争いと無縁な場所。「宮廷魔導師第三席アリア」ではなく、ただの「カフェ店主アリア」として再スタートできる場所を求めました。
